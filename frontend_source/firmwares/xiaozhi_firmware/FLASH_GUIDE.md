# מדריך צריבה - Xiaozhi AI Voice Robot (ESP32-S3 N16R8)

הפרויקט הזה הותאם ספציפית עבור המכשיר שלך עם ההגדרות הבאות:

- **צ'יפ:** ESP32-S3
- **זיכרון:** 16MB Flash, 8MB Octal PSRAM (N16R8)
- **מסך:** 0.96" OLED (SSD1306)
- **חיבור:** WiFi עם כתובת שרת מקומי מוגדרת מראש.

## הכנות על מחשב היעד (Ryzen AI)

1. וודא שמותקן **Python 3**.
2. התקן את כלי הצריבה:

   ```bash
   pip install esptool platformio
   ```

3. חבר את המכשיר למחשב. אם הפורט לא מזוהה, החזק את כפתור ה-**BOOT** ולחץ על **RESET**.

## צריבה באמצעות PlatformIO (מומלץ)

מתוך תיקיית הפרויקט, הרץ:

```bash
# בניית הפרויקט
pio run -e esp32-s3-devkitc-1

# צריבה
pio run -e esp32-s3-devkitc-1 -t upload
```

## צריבה ישירה (במידה ויש לך כבר קבצי Bin)

אם תרצה למחוק הכל ולצרוב מחדש (Erase Flash):

```bash
esptool.py --chip esp32s3 erase_flash
```

## הגדרות רשת מוגדרות מראש

הקוד כולל "הזרקה" אוטומטית של הגדרות לתוך ה-NVS בבוט הראשון. במידה ותרצה לשנות אותן לפני הצריבה, ערוך את קובץ `platformio.ini`:

- `FORCE_WIFI_SSID`: שם הרשת שלך
- `FORCE_WIFI_PASS`: סיסמת ה-WiFi
- `FORCE_WS_URL`: כתובת שרת ה-WebSocket שלך

## הערה לגבי גרסת ESP-IDF

הפרויקט דורש גרסה 5.5.0 ומעלה. הגדרות ה-Build הותאמו כדי לעקוף דרישות גרסה נוקשות של ספריות מסוימות.
